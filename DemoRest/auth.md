# Authorization, authentication, JWT.

author. authent. -> application security.

Authentication - verify user identity

Authorization - identify which resources user can access.

- Session-based (cookies)
- Token-based auth: login -> token (user, roles, expiry date)
- ACL, api gateways 


OAuth 2.0 - Open Authentication 

JWT - json web tokens.

3 parts:
- header: algorithm for hashing, type...
- payload: role(claims), expiration date, email, username ...
- signature: encoded using sercret key
