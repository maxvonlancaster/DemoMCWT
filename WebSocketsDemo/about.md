# WebSockets, WebHooks

## About

**WebSockets** — communication protocal for two-way, persistant communication between client and server in client-server architecture. 

Connection, stays open, continuos communication. 

Client <-> Server send messages in real time. 

Usages: chats, collaborative tools, internet-of-things monitorins. 

**WebHooks** - mechanism for servers to notify other systems (client, servers) for events. Automatic push. 

SignalR - nuget packege + js.
